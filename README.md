FEE - Electrical Engineering University of Campinas UNICAMP - FEE-247 (DEVELOPMENT OF IOT SOLUTIONS WITH LORA AND LORAWAN)

This Repository is an IoT Solution using Three-Phase Methodology for a LoRaWAN End Device ABP connecting on TTN over a 
Single Channel LoRaWAN Gateway, by MoT protocol using a Device PKLoRa-WAN by ESP32 core, LoRa radio RFM95, 
Sensors LDR & DHT22 & GPS
